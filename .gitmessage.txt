# 컨벤션
# Emoji/Type: <file name> - <commit message>

# Emoji/Type 가이드라인:
# ✨feat: 새로운 기능
# 🐛fix: 버그 수정
# 📚docs: 문서 수정
# 💄style: 코드 포맷팅, 세미콜론 누락 등 (비즈니스 로직 변경 없음)
# 🎨design: CSS 및 UI 수정
# ♻️refactor: 코드 리팩토링
# ✅test: 테스트 추가 또는 수정
# 🔧chore: 빌드 업무, 패키지 매니저 설정 등 (비즈니스 로직 변경 없음)
# 🖼️add: 이미지 등의 정적 자원 추가
# 🎉init: 초기 설정
# 🚚rename: 파일 또는 폴더명 수정 또는 이동

# 첫 설정 시 다음 코드 command에 복붙!
